# robot
Super cool robot 


Link for camera library: https://www.raspberrypi.com/documentation/computers/camera_software.html

Run UDP camera stream: rpicam-vid -t 0 --codec libav --libav-format mpegts -o "udp://192.168.2.182:1234"

Run video streamer: python videostream.py --video

Download ffmpeg windows from powershell: winget install Gyan.FFmpeg
